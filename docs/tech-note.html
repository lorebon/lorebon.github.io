Bonasera, L., Gualandi, S. (2024). Optimal shapelets tree for time series interpretable classification. EURO Journal on Computational Optimization, 12, 100091.

  • Possible extension: As defined in Eq. (1), the distance between a shapelet and a time series is computed by minimizing the distance between the shapelet and all subsequences of the time series having the same length. In the case of long shapelets, the magnitude of the distance may increase with the shapelet length, leading to larger Big-M values in Eqs. (60)–(61) and, consequently, to weaker linear relaxations.

    Given a shapelet \(\mathbf{y}\in[0,1]^\ell\) and a time series \(\mathbf{x}\in[0,1]^m\) with \(\ell < m\), one possible alternative is to use the normalized \(\ell_1\) distance

    \[ \operatorname{dist}(\mathbf{x}, \mathbf{y}) = \min_{1 \leq p \leq q} \frac{1}{\ell} \left\lVert \mathbf{y} - \mathbf{x}_p^\ell \right\rVert_1, \qquad q = m - \ell + 1. \]

    In this way, we can rewrite Eq. (73) as

    \[ M_i^L = \max \left\{ \frac{1}{\ell} \sum_{j\in[\ell]} \left(\theta_{ij}^+ + \theta_{ij}^- \right) - b \right\} \leq \max \left\{ \frac{1}{\ell} \sum_{j\in[\ell]} |s_j-u_{ij}| \right\} \leq 1, \qquad \forall i\in[n]. \]

    Moreover, the threshold variables can be bounded by replacing Eq. (71) with \[ 0 \leq b_t \leq 1, \qquad \forall t\in \mathcal{T}_B. \]

    Similarly, for Eq. (74) we obtain

    \[ M_i^R = \max\{b\} = 1, \qquad \forall i\in[n]. \]

Bonasera, L., Duma, D., Gualandi, S. (2024). Learning Sparse-Lets for Interpretable Classification of Event-interval Sequences. In Metaheuristics: MIC 2024 (Lecture Notes in Computer Science, Vol. 14754).

  • Possible extension: After being sampled, e-lets/sparse-lets are matched to each e-sequence using a sliding window. Therefore, Eq. (1) implies that the length of each e-let/sparse-let must be less than or equal to the length of the shortest e-sequence, that is

    \[ \ell(\mathcal{S}_i^{k, l}) \leq \min_{\mathcal{S}_j \in \mathcal{D}} \, \ell(\mathcal{S}_j), \qquad \forall k \in \{0, \dotsc, \ell(\mathcal{S}_i) - 1\}, \; \forall l \in \{1, \dotsc, \ell(\mathcal{S}_i) - k\}, \; \forall i \in \{1, \dotsc, N\}. \]

    This can be a limitation in applications where e-sequence lengths vary substantially. In such cases, the sampled e-lets/sparse-lets may be unable to capture relevant patterns in the data. One possible solution is to add a “no-event” symbol to the alphabet of the e-sequences. This makes it possible to pad shorter e-sequences with no-event symbols, thereby removing the constraint on the maximum length of e-lets/sparse-lets.
